This directory contains hardware wiring diagrams and schematics for the Arduino Car Library.

## Image Files to Add:

1. **single-motor-wiring.png** — Single motor setup with L298N
   - Arduino Uno connected to L298N module
   - Single DC motor configuration
   - For BasicCarControl.ino example

2. **dual-motor-wiring.png** — Dual motor setup with L298N  
   - Arduino Uno connected to L298N module
   - Two DC motors (left and right) configuration
   - For AdvancedNavigation.ino example on robot car

## How to Add Images:

1. Save diagrams as PNG or JPG format
2. Place in this `images/` directory
3. Commit and push to GitHub

Example filenames:
- `single-motor-wiring.png`
- `dual-motor-wiring.png`

The markdown files in README.md and docs/ will automatically display these images once they are added.
